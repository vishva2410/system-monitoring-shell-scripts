# System Monitoring Shell Scripts (macOS)

Lightweight Bash scripts for monitoring system RAM and disk usage on macOS using native system commands.

## Scripts
- ram_status.sh — Checks free RAM and alerts if memory is low
- ram_space.sh — RAM-related utility script
- disk_space.sh — Checks free disk space and alerts if it goes below a threshold

## Requirements
- macOS
- Bash

## Usage
```bash
chmod +x *.sh
./ram_status.sh
